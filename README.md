# 🍷 DineFlex

**Luxury Dining. Pay Later.**

DineFlex is the first *Buy Now, Pay Later* platform designed exclusively for luxury restaurants.  
We allow guests to enjoy premium cuisine today and pay in 4 interest-free installments.

---

## 🔥 Features

- 🍽️ Restaurant selection with premium dish photos  
- 🛍️ Smart cart with BNPL calculator  
- 💳 4-installment breakdown for each order  
- 👤 VIP Customer Profile with points and levels  
- 🌗 Dark mode toggle  
- 🎁 Reward actions: Invite a friend, Redeem points  

---
